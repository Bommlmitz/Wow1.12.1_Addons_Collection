Lions! v1 - Mark Murray

Simple addon that changes the Griffons on the ends of your main menu bar to Lions.

Install:

Extract the contents of this folder to your interface folder. Directory should be

World of Warcraft\Interface\MainMenuBar

Remove:
Delete this folder
