The mod takes into account the following conditions.

If you have a target.
If your target is dead.
If your target is humanoid, beast or critter.
If your target is friendly.
If your target is a world boss.
If you are currently casting.
If your Polymorph macro is on cooldown.
If Polymorph is in range.
If any party member is mind controlled.
If you are in a disallowed zone.
If you are set to never emote with the current target.
If you are set to always emote with the current target.
