If you like this mod please consider making a donation at http://wow.pchjaelp.dk

Installation:
Put the DeamonTrainerFrame folder into the \World of Warcraft\Interface\AddOns\ directory.

Useage:
This frame will show up at the demon trainer instead of the normal merchant frame.
Type /dtf or /DemonTrainerFrame to see a list of slashcomamnds
