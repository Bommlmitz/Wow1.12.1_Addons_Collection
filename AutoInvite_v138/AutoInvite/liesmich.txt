# AutoInvite Mod für World of Warcraft
# Matthias Fechner
# idefix@fechner.net

Mit dem Mod ist es möglich auf einen Knopfdruck feste Gruppenzusammenstellungen einzuladen. Ich habe dieses Mod geschrieben um das einladen unserer Standardgruppe für die Raids zu vereinfachen. Es ist auch möglich Spieler mit Hilfe einer CSV Liste und Copy-and-Paste in das Mod einzufügen. Diese CSV Liste kann z.B. mit einem Tool eurer Wahl erstellt werden.
Alle Zusammenstellungen können gespeichert und geladen werden.
Das Mod ist auch in der Lage die Mitspielere automatisch in die entspechenden Gruppen zu verschieben (hier brauche ich noch Hilfe zum testen).


Das Hauptfenster kann mit dem Befehl /ai show geöffnet werden oder mit der Taste die man unter der Tastenbelgung festlegen kann.

Im ersten Schritt müssen Spieler in die Liste hinzugefügt werden, das kann auf mehrer Arten passieren:
1. Import der Spieler mit einer CSV Liste.
2. Einen Spieler anklicken und dann entweder in der GUI auf den Knopf Add klicken oder wenn die Tastenbelegungen eingestellt sind, diese Taste drücken.
3. Auf den Knopf Add drücken ohne einen Spieler ausgewählt zu haben, es geht ein Fenster auf in dem man den Namen eingeben kann.
4. Das Mod mit dem Hacken in der GUI aktivieren und einfach die Leute einladen mit /invite, über die Freundesliste oder Gilde etc.

Die Leute tauchen dann in der Liste ganz rechts auf und können mit einen Linksklick in die aktuellen Gruppenzusammenstellung verschoben werden, bei einen nochmaligen Klick mit links in den Tabellen werden die Leute wieder in die Liste ganz rechts verschoben.
Leute aus der rechten Liste lassen sich mit einem rechtsklick permanent entfernen.

Falls noch keine Informationen über den Spieler vorhanden sind, landen diese automatisch unter Druiden, werden aber wenn es möglich ist in die entsprechenden Tabellen verschoben (mit einem Klick auf Update kann man diesen Vorgang manuell anstossen).

Um die Leute nun einzuladen gibt es wieder mehrer Möglichkeiten:
1. Als Gruppe einladen, es wird eine normale Gruppe erstellt.
2. Als Gruppe einladen, indem man den Tastkürzel (der vorher eingestellt weden muss) gedrückt wird.
3. Als Raid einladen, es wird als Raid eingeladen.

Es besteht auch die Möglichkeit die Gruppenzusammenstellung zu speichern und das Tool dann automatisch die Gruppen umstellen zu lassen.
Hierfür können die Leute mit einem Rechtsklick in den linken Tabellen in Gruppen eingeteilt werden oder man verschiebt sie einfach im Schlachtzugfenster und klickt dann auf Read Groups.
Das verschieben der Leute geschieht dann mit Move Groups (funktioniert momentan noch nicht).

Die aktuelle Zusammenstellung kann mit Load-Save abgespeichert oder geladen werden.

Die Farbkodierung in den linken Tabellen ist die folgende:
Grau = noch nicht eingeladen
Gelb = eingeladen
Grün = ist der Gruppe beigetreten
Rot = hat Einladung abgelehnt oder der Spieler ignoriert einen
Blau = Spieler ist noch in Gruppe (er bekommt automatisch einen Wisper, wenn der Hacken in der GUI gesetzt ist)

Laden von Spielern mittels CSV Datei:
*************************************
Die Felder sind:
name:eClass:level:inGroup:group:comment

Für eine genauere Beschreibung der Felder siehe weiter unten.

Beispiel:
Player1:PRIEST:60:1:-:Test..
Player2:HUNTER:60:0:1:bla<br>Zeile2<br>Zeile3

Dann eine kurze Erklärung der Felder:
name: Name des Spielers
group: Nummer der Gruppe (1 bis 8), "-" entspricht noch keiner Gruppe zugeteilt
eClass: hier muss PRIEST, HUNTER, WARRIOR, MAGE, PALADIN, SHAMAN, WARLOCK, ROGUE oder DRUID stehen
inGroup: true = Spieler soll eingeladen werden, false = Spieler soll nicht eingeladen werden aber in der rechten Liste auftauchen
level: kann man auf 0 stehen lassen, das mod liesst das Level dann automatisch ein

Dann öffne das Menü, drücke Save-Load -> Import CVS -> Gib einen Namen für das Profil ein und kopiere die CSV Daten mit copy and paste -> import
