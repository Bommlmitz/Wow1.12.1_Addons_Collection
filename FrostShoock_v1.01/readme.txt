--[ INSTALLATION INSTRUCTIONS ]-----------------------------------------------------

Extract the folder FROSTSHOOCK to your "World of Warcraft/Interface/Addons/" folder
on your computer.  Run World of Warcraft, click the "Addons" button in the Character
Selection screen, and check the box labeled "FROSTSHOOOOOCK!!!!"


--[ USING THE ADDON ]---------------------------------------------------------------

FROSTSHOOCK is disabled upon login.

To switch between enabled and disabled, simply type "/frostshoock", or "/fs".

Type "/fs help" to display usage information
