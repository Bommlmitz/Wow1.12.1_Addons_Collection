myMemory v1.2


Description
===========

myMemory is a World of Warcraft AddOn. It's a memory usage display.


Install
=======

Extract the files into your ..\World of Warcraft\Interface\AddOns directory.


Features
========

* Memory usage display
* Low memory usage (~0.01MB)
* myAddOns support


Usage
=====

The memory usage is displayed just above the framerate when it's visible. You can
toggle the display with the framerate toggle key binding (CTRL+R by default).
